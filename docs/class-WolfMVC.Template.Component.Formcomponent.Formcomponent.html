Class Formcomponent

Classe astratta base di tutti i form component.
I suoi attributi corrispondono agli attributi dei tag html interni ad un form. Presenta il metodo astratto make($html) che scrive il codice html del campo e il metodo non astratto attributes2html($html) che compila gli attributi applicabili al tag in questione.
