# __call( strin $name, array $arguments )

Override del metodo magico __call. Controlla se il metodo richiesto può essere interpretato come un getter o un setter per la classe corrente, ovvero se esiste la proprietà richiesta dal metodo. Nel primo caso agisce solo se la proprietà è leggibile, nel secondo agisce solo se la proprietà è scrivibile. Nel caso peggiore lancia eccezione.
